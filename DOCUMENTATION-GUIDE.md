## üîç Quick Reference

### Architecture Components
- **Server**: Node.js + Express.js
- **Real-Time**: WebSocket (ws)
- **File Watching**: Chokidar
- **Image Processing**: Sharp + Canvas
- **Security**: bcryptjs + Express-Session
- **Uploads**: Multer

### Key Features
- **Authentication**: Session-based + JWT tokens
- **Real-Time**: WebSocket bi-directional communication
- **Image Optimization**: Automatic resizing and format conversion
- **File Management**: Copy, move, rename, delete, upload
- **AI Features**: OCR, color extraction, duplicate detection
- **Analytics**: Usage tracking and visualization

### User Roles
1. **Admin**: Full control
2. **Manager**: Content management
3. **Operator**: View only

### Applications
1. **Daily Plan Viewer**: Time-based schedule display
2. **Image Gallery**: Browse, slideshow, meeting modes
3. **Performance Dashboard**: KPI tracking with drag-and-drop

## To Make Repository Public and Visible:

### Option 1: Via GitHub Website

1. Go to: https://github.com/aleks131/Bachelor-Project-
2. Click on **"Settings"** tab (top right of repository page)
3. Scroll down to **"Danger Zone"** section
4. Click **"Change visibility"**
5. Select **"Make public"**
6. Type repository name to confirm
7. Click **"I understand, change repository visibility"**

### Option 2: Via GitHub CLI (if installed)

```bash
gh repo edit aleks131/Bachelor-Project- --visibility public
```
